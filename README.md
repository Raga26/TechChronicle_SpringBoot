# BlogApp_SpringBoot

High Level Requirment :

1. Post Management
2. Comments Management
3. Authentication and Authorization
4. Category Management
----------------------------------------------------------------

1. Post Management
   - Create post
   - get single post
   - get all posts
   - get all posts with pagination and Sorting
   - update post
   - delete post

  2. Comments Management (one to many mapping)
     - Create comment
     - get single comment
     - get all comments
     - update comment
     - Delete comment

  4. Authentication and authorisation
     - login/logout
     -  Role based security (Admin/User)

  5. Category management
     - Create category
     - get single category
     - get all categories
     - update category
     - delete category
     - get posts by category

Deployment on AWS.
